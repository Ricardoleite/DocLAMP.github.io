Resultados Numéricos: MHD - 1 Bloco

Abaixo são apresentados os resultados de Nusselt médio obtidos em função do número de Rayleigh.

Constantes de Ajuste

Com base nos resultados numéricos as constantes \(C_{0}\), \(C_{1}\) e \(C_{2}\) são ajustadas para o caso de 1 bloco. Dado a correlação obtida

\[ \overline{Nu}= \displaystyle\frac{L}{H} \gamma^{-1/2} =\displaystyle\frac{L}{H} \left( C_{1} \left(Ra_{H}\left(\displaystyle\frac{Pr}{C_{0}+Pr}\right)\right)^{-1/2} -C_{2} Ha^{2} Ra_{H}^{-1} \right)^{-1/2}, \]

as constantes \(C_{0}\) e \(C_{1}\) são ajustados com base nos resultados obtidos \(Ha=0\) e a constante \(C_{2}\) como uma média dos valores ajustados para \(Ha=50\) e \(Ha=100\);

Ajuste de \(C_{0}\) e \(C_{1}\) para Ha=0

Com base no ajuste feito foram obtidos \(C_{0}=0,112888\) e \(C_{1}=0,00649\). A figura abaixo ilustra o ajuste numérico.

import matplotlib
import matplotlib.pyplot as plt
import numpy as np
N1=np.array([3.59,3.88,4.21,4.27,6.95,7.44,8.5,8.74])
gamma=np.ones((len(Ra),len(Pr)))
# A1=0.004496
# B1=0.0051449
A1=0.00649
B1=0.112888
for j in range (len(Ra)):
    for i in range (len(Pr)):
        gamma[j][i]=(A1*Ra[j]*Pr[i]/(B1+Pr[i]))**(-1./2.)
gama=gamma.reshape(len(Pr)*len(Ra))

plt.semilogx(gama,N1,'ro',fillstyle='none')
CN1=gama
plt.semilogx(gama,CN1**(-1./2.),'r--',fillstyle='none')
plt.ylabel('$Nu_{av}$', color='k', horizontalalignment='right',rotation=0, fontsize=20)
plt.xlabel('$\\gamma$', fontsize=20)
plt.legend(['LBM','Correlação'],fontsize=11.,bbox_to_anchor=(1.0, 1.02))
plt.show()

Ajuste de \(C_{2}\) para Ha=50

Com base no ajuste feito foi obtidos \(C_{2}=-5,1\). A figura abaixo ilustra o ajuste numérico.

Ajuste de \(C_{2}\) para Ha=100

Com base no ajuste feito foi obtidos \(C_{2}=-20,9\). A figura abaixo ilustra o ajuste numérico.

Correlação Ajustada

Tirando a média de \(C_{2}\), tem-se \(C_{2}\)=-5,5. Portanto, a correlação ajustada é dada por

\[ \overline{Nu}= \displaystyle\frac{L}{H} \gamma^{-1/2} =\displaystyle\frac{L}{H} \left( 0,00649 \left(Ra_{H}\left(\displaystyle\frac{Pr}{0,112888 +Pr}\right)\right)^{-1/2} +5,5 Ha^{2} Ra_{H}^{-1} \right)^{-1/2}. \]
